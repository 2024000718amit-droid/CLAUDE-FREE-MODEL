# Unlimited Claude Putter

A beautiful, fully functional Claude.ai clone — **completely free & unlimited** using Puter.js v2.

No API keys. No backend. No daily limits. Just pure Claude Opus 4.6, Sonnet 4.6 & Haiku 4.5 in your browser.

![Preview](https://github.com/yourusername/unlimited-claude-putter/blob/main/preview.png)

## ✨ Features
- Modern Claude.ai-like UI (dark mode by default)
- Real streaming responses
- Full chat history (saved in browser)
- Model selector (Opus 4.6 • Sonnet 4.6 • Haiku 4.5)
- Markdown + code syntax highlighting
- Copy, Regenerate, Delete chats
- Responsive (mobile + desktop)
- One-click Puter authentication
- Keyboard shortcut (Ctrl/Cmd + Enter)

## 🚀 Quick Start
1. Download or clone this repo
2. Open `index.html` in your browser
3. First message will trigger Puter login (free & instant)
4. Start chatting with unlimited Claude!

Or use GitHub Pages for instant online access.

Made with ❤️ using [Puter.js](https://puter.com)
